# Release v0.1.3 (TaskLyn)

This update brings a more professional look and a fully automated update system to TaskLyn.

### ✨ What's New
- **Professional Update System**: No more manual installer downloads! The app now detects, downloads, and prompts you to install updates automatically via GitHub.
- **Improved Settings UI**: A new "Updates" section has been added to give you real-time feedback on your app's version and update status.
- **Fluid Task Animations**: Switching filters (All, Pending, Completed) or searching now features smooth, staggered layout animations.

### 🐛 Bug Fixes & Refinements
- **Ghosting Fix**: Completed tasks no longer appear transparent or show "ghosting" effects when being dragged and sorted.
- **Design Consistency**: In dark mode, dragging a completed task now preserves its "muted" look while remaining 100% opaque.
- **Sorting Reliability**: Fixed a bug where Framer Motion transitions would sometimes interfere with the accuracy of the Drag-and-Drop system.

### 🛠️ Developer Improvements
- **Automated CI/CD**: Added GitHub Actions to handle automated builds and releases whenever a tag is pushed.
